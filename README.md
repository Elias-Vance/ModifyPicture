# PictureModify-图片修改器(Java桌面端)



A tool project focused on image processing, offering convenient image editing features to help users easily adjust image effects.

## Features

There are many filters available, such as binarization, mosaic, negative, oil painting, contour drawing, color conversion, and more.

## Usage

The files in the `src` folder are Java source code. The `main` method serves as the entry point to launch the program. It is recommended to directly copy these source files into IntelliJ IDEA (or your preferred IDE) to run the project.
